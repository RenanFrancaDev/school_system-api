# School Classroom Management API

A complete REST API for managing classrooms, students, teachers and allocations built with AdonisJS 6.

## Features

- Student and teacher management
- Classroom allocation system
- JWT Authentication
- Capacity control and availability
- MySQL database with Lucid ORM

## Tech Stack

- AdonisJS 6
- TypeScript
- MySQL
- JWT Authentication
- Lucid ORM
